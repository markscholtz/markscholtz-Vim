README - dotvim
===============

Setup
-----

* git clone git://github.com/markscholtz/markscholtz-Vim.git .vim
* `./install.rb`

What it does
------------

* `ln -s ~/.vim/vimrc ~/.vimrc`
* `git submodule update --init`
* `rake make` in bundle/command-t

Dependencies
------------

* rake
* ruby
* rvm
